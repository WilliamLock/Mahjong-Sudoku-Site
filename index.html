Classic rules, different visual challenge

You still solve each row, column, and 3x3 area with one of each value from 1 to 9. The difference is that the board can use Mahjong-style suits like dots, bamboo, and characters, turning a familiar number puzzle into a more visual pattern challenge.

Find it on the App Store

Search for Mahjong Sudoku Classic. Look for the light Sudoku board icon with a Mahjong tile and colored dots. Developer: William Lock.
