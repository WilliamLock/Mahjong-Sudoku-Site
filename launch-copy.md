## Reddit / Developer Forum Post

I built Mahjong Sudoku Classic, a small puzzle game that combines classic Sudoku logic with Mahjong-inspired tile suits.

The rule system is still Sudoku: each row, column, and 3x3 area needs one of each tile from 1 to 9. The change is visual. Instead of a plain number grid, the board uses Mahjong-style suits like dots, bamboo, and characters.

I wanted to make Sudoku feel a little warmer and more cultural without changing the core logic that makes the puzzle work.

正文：
我做了一个小 puzzle app，叫 Mahjong Sudoku Classic / 麻将数独。

它的规则还是大家熟悉的数独：每一行、每一列、每一个 3x3 区域，都要放入 1 到 9 的不同元素。

但我想让它的感觉不只是普通数字格子，所以把视觉改成了麻将牌风格。你可以用筒、索、萬这些更有东方感的牌面来玩数独。

我喜欢这种小而安静的游戏：不用账号，不需要复杂系统，打开就可以慢慢解题。

如果你平时喜欢数独、麻将、或者安静一点的益智游戏，可以试试看。
