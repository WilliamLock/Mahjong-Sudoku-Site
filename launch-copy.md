## Short English Post

I made Mahjong Sudoku Classic because I’m a Sudoku fan, and I wanted to see what would happen if the usual 1-9 grid became Mahjong tiles.

The rules are still classic Sudoku: every row, column, and 3x3 area needs one of each value. But instead of only reading digits, you solve with Mahjong-style tile suits like dots, bamboo, and characters.

## Reddit / Developer Forum Post

I built Mahjong Sudoku Classic because I’m a Sudoku fan, and I wanted to combine single-solution Sudoku logic with Chinese Mahjong tile visuals.

When solving normal Sudoku with numbers 1-9, some puzzles can get difficult in a very abstract way. I started wondering whether replacing the numbers with Mahjong-style tiles would make the board feel more visual and more challenging.

The rule system is still Sudoku: each row, column, and 3x3 area needs one of each tile from 1 to 9. The difference is the visual language. Instead of a plain number grid, the board can use Mahjong suits like dots, bamboo, and characters.

I also liked the idea of connecting Mahjong players with people who enjoy brainy logic games.

## Xiaohongshu / Rednote

标题：
我是数独迷，所以做了一个麻将数独

正文：
我本身很喜欢玩数独。数独最吸引我的地方，是它看起来很简单：1 到 9，每一行、每一列、每个 3x3 区域都不能重复。

但玩久了之后也会发现，有些单解的数独题到后面会变得很难，全部都是 1-9 的数字，有时候看起来也会有点抽象。

后来我想到一个问题：如果用中国麻将牌来代表这些数字，会不会让数独变得更有趣，也更有挑战？

所以我做了 Mahjong Sudoku Classic / 麻将数独。

它的规则还是数独：每一行、每一列、每个 3x3 区域都要放入 1 到 9 的不同元素。但视觉上可以用筒、索、萬这些麻将牌面来解题。

我觉得这个 idea 很有意思，因为它把喜欢麻将的人，和喜欢数独这种动脑游戏的人连接在一起。

如果你平时喜欢数独、麻将、或者安静一点的益智游戏，可以试试看，也欢迎告诉我：麻将牌面是让数独更好玩，还是更难？
